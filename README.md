# Bitcoin: Not Crypto - Reveal.js Presentation

A professional presentation built with reveal.js that explores the fundamental differences between Bitcoin and other cryptocurrencies.

## Features

- **Modern Design**: Clean, professional layout with Bitcoin-themed styling
- **Interactive Slides**: Smooth transitions and fragment animations
- **Responsive**: Works on desktop, tablet, and mobile devices
- **Accessible**: Keyboard navigation and screen reader friendly
- **Custom Styling**: Bitcoin orange color scheme with custom CSS

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn package manager

### Installation

1. Clone or download this repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the Presentation

You have several options to run the presentation:

#### Option 1: Development Server (Recommended)

```bash
npm run dev
```

This will start a live server with auto-reload at `http://localhost:8080`

#### Option 2: Simple HTTP Server

```bash
npm run serve
```

This will serve the presentation at `http://localhost:8080`

#### Option 3: Direct File Access

Simply open `index.html` in your web browser.

## Presentation Structure

The presentation covers the following topics:

1. **Introduction** - Title slide with Bitcoin branding
2. **Agenda** - Overview of presentation topics
3. **What is Bitcoin?** - Core principles and characteristics
4. **What is "Crypto"?** - Broad category of digital assets
5. **Key Differences** - Comparative analysis
6. **Why Bitcoin Stands Apart** - Unique advantages
7. **The Future** - Vision for digital money
8. **Conclusion** - Key takeaways

## Customization

### Styling

- Edit `styles.css` to modify colors, fonts, and layout
- The main Bitcoin color is `#f7931a` (Bitcoin orange)
- Font used is Inter from Google Fonts

### Content

- Edit `index.html` to modify slide content
- Add new slides by creating new `<section>` elements
- Use fragments for animated content reveals

### Configuration

- Modify the Reveal.js configuration in the `<script>` section at the bottom of `index.html`
- Adjust transition effects, themes, and behavior

## Keyboard Controls

- **Arrow Keys**: Navigate between slides
- **Space**: Next slide
- **Shift + Space**: Previous slide
- **Home**: First slide
- **End**: Last slide
- **F**: Fullscreen mode
- **Esc**: Overview mode
- **S**: Speaker notes (if enabled)
